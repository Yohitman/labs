ставим докер на убунту
apt-get install docker
apt-get install docker.io

добавляем докер в группу исключений
sudo usermod -a -G docker pavel

запуск нового контейнера
docker run -it alpine:3.6

обновление пакетов
apk update
apk upgrade

скачивание пакетов приложений
apk add nano
apk add gcc

Cоздадим каталог "work" и переходим в него:

mkdir work

cd work

Копируем файлы с удаленного сервера:

scp -r -P 6666 student@openit.guap.ru:/container/ABC-Linux/lab4/* work/

правим файлы

коммитим докер с другого терминала
docker ps
docker commit /номер из ps/ /папка в докер хабе куда скидывать/
